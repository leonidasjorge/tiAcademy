1. Qual é a diferença entre as variáveis: const, let e var?
Resposta: A principal diferença é o escopo onde elas são declaradas.
Dentro de funções, as variáveis são sempre locais.
As variáveis "const e let" sempre vão depender do escopo; elas podem ser globais mas se tiverem dentro de um escopo, sempre vão funcionar somente dentro daquele escopo.

2. É verdade que o uso de parâmetros em funções é obrigatório?
Resposta: Não é verdade. Só se usa parâmetros se tiver necessidade.

3. Selecione a opção verdadeira para inserir um arquivo JS em um documento HTML:
a) <link src="nomeDoArquivo.js"></link>
b) <meta src="nomeDoArquivo.js"></meta>
c) <script src="nomeDoArquivo.js"></script>
d) <a href="nomeDoArquivo.js"></a>

Resposta: c) <script src="nomeDoArquivo.js"></script>
