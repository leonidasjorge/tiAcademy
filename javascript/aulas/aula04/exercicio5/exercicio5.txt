Exercício 5:

1. Dê um exemplo de uma variável obedecendo a boa prática "camelCase".
Resposta: var NomeCliente;

2. Por padrão, qual o tipo de dado atribuído a uma variável quando esta não recebe nenhum valor?
Resposta: undefined (indefinido);

3. Quais são as duas categorias de tipos de dados do JavaScript (JS)?
Resposta: Tipo "Primitivo" e "Não Primitivo";

4. É verdade que o JavaScript é uma linguagem "server-side"? Explique.
Resposta: Não é verdade. O JavaScript é uma linguagem "client-side" (do lado do cliente). No entanto, conseguimos sim fazer requisições HTTP com o JavaScript.
